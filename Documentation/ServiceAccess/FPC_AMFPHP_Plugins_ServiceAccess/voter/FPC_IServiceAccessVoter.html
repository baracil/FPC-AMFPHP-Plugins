Interface FPC_IServiceAccessVoter

Description
Description | Methods (details)

An voter interface that grants or denies access to a service.

Before calling the service, the voter is called with information about the user calling the service, the called service object and the parameters of the called service. The voter should return true if the user trying to access the service is allowed to do so.
