Class FPC_RolesSAVoter

Description
Description | Methods (details)

Implements interfaces:

  • FPC_IServiceAccessVoter

A voter that checks the roles of the user

If the calling user has at least one of the allowedRoles (set at construction) then the access is granted. Otherwise the access is denied.
