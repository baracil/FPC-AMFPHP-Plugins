Class FPC_MethodRolesSAVoterProvider

Description
Description | Methods (details) Constants (details)

Implements interfaces:

  • FPC_IServiceAccessVoterProvider

A provider that returns a FPC_RolesSAVoter

The returned voter is initialized with the roles returned by a given method (by default 'getMethodRoles'). The method name can be set at construction.

The signature of this method should be

  1.   public function getMethodRoles($methodName) {
  2.    //do some work
  3.    return AN_ARRAY_OF_STRING_LISTING_THE_ROLES_ALLOWED_FOR_THE_METHOD_METHODNAME;
  4.  }
