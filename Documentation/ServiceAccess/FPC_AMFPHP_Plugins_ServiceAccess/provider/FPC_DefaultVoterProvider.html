Class FPC_DefaultVoterProvider

Description
Description | Methods (details)

The default voter provider.

It is a FPC_ComitySAVoterProvider that handles @rolesAllowed (see FPC_ReflectionRolesSAVoterProvider), @isCurrentUserLogin (see FPC_CurrentUserLoginSAVoterProvider) and @methodCheck (see FPC_ReflectionMethodSAVoterProvider) annotations. By default, this provider caches the data into the SESSION. This can be changed by passing false to the constructor (useful when developing).

see the main documentation (in ServiceAccess) for more details
