Class FPC_CachedSAVoterProvider

Description
Description | Methods (details) Constants (details)

Implements interfaces:

  • FPC_IServiceAccessVoterProvider

A provider that caches its results

Since a voter is always the same for a method, the result of a provider can be cached. This provider does exactly that. The voters are cached in the SESSION under the key FPC_CachedSAVoterProvider::CACHED_SA_VOTER_SESSION_KEY
