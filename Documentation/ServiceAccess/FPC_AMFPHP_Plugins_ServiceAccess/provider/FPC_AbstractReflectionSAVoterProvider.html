Class FPC_AbstractReflectionSAVoterProvider

Description
Description | Descendents | Methods (details)

Implements interfaces:

  • FPC_IServiceAccessVoterProvider

Abstract class used for providers using "annotations". The class extending it must set the $tag value at construction.

The $tag value is the annotation name searched in the documentation. The $tag does not include the beginning "@".

When called, the documentation of the method is searched for the $tag. If found, the values following it are sent to the abstract method 'handleValues'. Values must be separated by a coma. For instance

  1.    /**
  2.     * Some function documentation
  3.     * @MyTag value1, value2, value3
  4.     */
  5.     public function aFunction() {...}

By default, if multiple tags are present in the documentation only the first one is used, for instance :

  1.    /**
  2.     * Some function documentation
  3.     * @MyTag value1, value2, value3
  4.     * @MyTag value4, value5, value6
  5.     */
  6.     public function aFunction() {...}

Only value1, value2 and value3 will be used. This behaviour can be changed by setting the secondary parameter of the constructor to 'true'. Then values are merged. So with the previous example, value1, value2, value3, value4, value5 and value6 will be used.
