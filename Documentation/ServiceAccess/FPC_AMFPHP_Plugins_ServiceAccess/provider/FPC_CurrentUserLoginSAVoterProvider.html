A provider that create a FPC_CurrentUserLoginSAVoter.

The provider uses the @isCurrentUserLogin annotation to get the two parameters need by the constructor of the FPC_CurrentUserLoginSAVoter. The syntax is :

  1.   /**
  2.    * @isCurrentUserLogin [index[,loginProperty]]
  3.    */
