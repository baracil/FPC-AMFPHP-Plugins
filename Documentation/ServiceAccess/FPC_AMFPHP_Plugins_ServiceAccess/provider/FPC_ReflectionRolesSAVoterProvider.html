A provider that returns a FPC_RolesSAVoter

The provider uses the values of the annotation @rolesAllowed to initialize the voter. For instance, with :

  1.    /**
  2.     * @rolesAllowed ADMIN, MODERATOR
  3.     */
  4.     public function securedMethod() {...}

The method 'securedMethod' can only be called by user with at least the role ADMIN or MODERATOR.

If the documentation contains several @rolesAllowed annotations, the values of each occurences are merged. So

  1.   /**
  2.    * @rolesAllowed ADMIN, MODERATOR
  3.    * @rolesAllowed USER
  4.    */
  5.    public function securedMethod() {...}

is exactly the same than

  1.   /**
  2.    * @rolesAllowed ADMIN, MODERATOR, USER
  3.    */
  4.    public function securedMethod() {...}
