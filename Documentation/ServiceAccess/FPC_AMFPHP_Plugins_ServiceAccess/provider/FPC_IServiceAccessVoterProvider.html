Interface FPC_IServiceAccessVoterProvider

Description
Description | Methods (details)

A voter provider interface that provides the FPC_IServiceAccessVoter associated to a method.

ServiceAccess plugin must be configured with an implementation of this interface. It is used to determine the voter associated with a method.

By default, the plugin uses FPC_DefaultVoterProvider.
